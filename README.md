# Pets Planet

E-Commerce Application.

## Packages used
* Google fonts
* Image picker
* Shared Preferences
* Hive
* Flutter Bloc 
* Firebase
* Cached Network Image
* Cloud Firestore
* Connectivity Plus
* Equatable
* Firebase Core
* Firebase Storage
* Flutter Screenutil
